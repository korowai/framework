Rfc8089
=======

The class :class:`Korowai\\Lib\\Rfc\\Rfc8089` provides rules for parsing
URIs according to file URI scheme, as defined in RFC8089_. See API docs of
:class:`Korowai\\Lib\\Rfc\\Rfc8089`.
