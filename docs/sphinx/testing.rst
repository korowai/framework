Testing
=======

:ref:`TheKorowaiFramework` provides *korowai/testing* package to facilitate
implementation of unit tests when developing other packages. It's based on
PHPUnit_ by Sebastian Bergmann and defines several extra assertions and
additional methods.

Installation
------------

.. code-block:: shell

   php composer.phar require "korowai/testing:dev-master"


Purpose
-------

This package provides extensions to PHPUnit_.
